# スキルガイド

各スキルの実践ガイドです。具体的な使用例、ワークフローの解説、活用のコツを含みます。

スキルガイドでは以下の構成で各スキルを詳しく解説しています：

1. **概要** - 何ができるか、どんな問題を解決するか
2. **前提条件** - 必要なAPIキー、Pythonパッケージ
3. **クイックスタート** - 最短で試す方法
4. **仕組み** - 内部ワークフローの解説
5. **使用例** - 具体的なプロンプトと期待される結果
6. **出力の読み方** - レポートの解釈方法
7. **Tips & ベストプラクティス** - パワーユーザー向けの活用法
8. **他スキルとの連携** - マルチスキルワークフロー
9. **トラブルシューティング** - よくある問題と対処法
10. **リファレンス** - CLIオプション一覧

> 検索は英語スキル名（"CANSLIM", "VCP", "FinViz"等）での検索を推奨します。日本語の部分一致検索は制限があります。
